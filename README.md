# CRUD de Usuarios en PHP

## 📌 Descripción
Sistema CRUD desarrollado en PHP y MySQL que permite registrar, listar, editar y eliminar usuarios.

## ⚙️ Tecnologías utilizadas
- PHP
- MySQL
- XAMPP

## 🚀 Funcionalidades
- Crear usuarios
- Listar usuarios
- Editar usuarios
- Eliminar usuarios

## 🛠️ Instalación
1. Clonar el repositorio
2. Importar la base de datos (archivo .sql)
3. Ejecutar en localhost con XAMPP
