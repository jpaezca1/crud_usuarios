# 🚀 Sistema CRUD de Usuarios con PHP y MySQL

## 📌 Descripción

Este proyecto es una aplicación CRUD (Crear, Leer, Actualizar y Eliminar) desarrollada en PHP y MySQL, que permite gestionar usuarios con nombre, correo y teléfono.

## 🧠 ¿Qué es un CRUD?

CRUD significa:

* Crear (Create)
* Leer (Read)
* Actualizar (Update)
* Eliminar (Delete)

## 🛠️ Tecnologías utilizadas

* PHP
* MySQL
* XAMPP
* HTML

## 📂 Estructura del proyecto

```
crud_usuarios/
│── conexion.php
│── Usuario.php
│── index.php
│── listar.php
│── editar.php
│── eliminar.php
│── crud_php.sql
│── README.md
```

## ⚙️ Instalación

1. Clonar el repositorio:

```
git clone https://github.com/jpaezca1/crud_usuarios.git
```

2. Copiar en la carpeta `htdocs` de XAMPP

3. Importar la base de datos:

* Abrir phpMyAdmin
* Crear base de datos `crud_php`
* Importar archivo `crud_php.sql`

4. Iniciar Apache y MySQL

5. Abrir en el navegador:

```
http://localhost/crud_usuarios
```
